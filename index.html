Projects

Project 1: Forecasting Loss of Signal in Optical Networks with Machine Learning

Developed a supervised ML approach for forecasting Imminent Loss of Signal (ILOS) in optical networks 1-7 days in advance. Tailored ML techniques and analyzed PM data from six commercial networks, encompassing 1,373 days and 16,456,331 data samples. Achieved 3% ILOS event forecast with 65% precision network-wide, and over 80% precision for 100G OTN lines, albeit with lower recall. A single ML model offers nearly optimal performance across all networks, facilitating easy integration into commercial products for proactive network management and outage prevention.

Project 2: Converting Speech Input into Text using Natural Language Processing

Developed a speech recognition application in Python to convert audio speech input to text output. Built custom deep neural network architecture leveraging TensorFlow and Keras libraries to train acoustic and language models on over 50 hours of human speech data. Achieved over 85% accuracy in transcription of audio to text for common vocabulary. Optimized TensorFlow model deployment on a cloud platform to create a scalable API, allowing web and mobile integration for applying speech-to-text in a variety of human-computer interaction systems.
